======================================================
  如何管理照片 / How to Manage Photos
======================================================

【文件夹结构 / Folder Structure】

  photos/
  ├── hero/               ← 首页幻灯片照片
  │   └── thumbs/         ← (可选) 低分辨率预览版
  │
  ├── portraits/          ← 人像照片（高清）
  │   └── thumbs/         ← (可选) 低分辨率预览版
  │
  ├── still-life/         ← 静物照片（高清）
  │   └── thumbs/         ← (可选) 低分辨率预览版
  │
  ├── landscapes/         ← 风景照片（高清）
  │   └── thumbs/         ← (可选) 低分辨率预览版
  │
  └── wechat-qr.jpg       ← 微信二维码名片图片

【添加照片 / Adding Photos】
1. 将高清照片放入对应分类文件夹。
2. 将同名的低分辨率版本放入对应 thumbs/ 子文件夹。
   （若无低分辨率版本，网站会自动使用高清版预览。）
3. 重新构建项目（npm run build）即可更新网站。

【支持格式 / Supported Formats】
  .jpg  .jpeg  .png  .webp
